## Setup

Make sure to install the dependencies:

```bash
# pnpm (recommended)
pnpm install

# yarn
yarn install

# npm
npm install
```

## Development Server

Start the development server on [localhost:3000](http://localhost:3000)

```zsh
pnpm run dev
```

## Production

Build the application for production:

```zsh
pnpm run build
```

Locally preview production build:

```zsh
pnpm run preview
```
